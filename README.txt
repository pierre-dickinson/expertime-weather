Expertime Weather plugin display a “Weather” menu in the WordPress back office and a weather search page accessible through a custom url.

== Description ==

Result page display weather of the day regarding position or address sent by the user. It also displays the forecast for the next 4 coming days.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `expertime-weather.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your api endpoint url and google api key from the admin plugin setting section
